## KendoCalendar Widget for [Wakanda](http://wakanda.org)
The __KendoCalendar__ widget allows you to select a date.
See it in [action](http://demos.telerik.com/kendo-ui/web/calendar/index.html).

Before using this widget, you must have installed [WakendoCore](https://github.com/acoudeyras/WakendoCore)


### Properties
This widget has the following properties:

* __value__: The date selected.
